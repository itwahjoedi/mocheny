# Mocheny

**Declarative Environment Parity & Secret Lifecycle for Hybrid PaaS**

Mocheny is a zero-trust control plane for managing **State**, **Parity**, and **Secrets** across Jelastic, Cloudflare Workers, and containerized environments.

> Official source: [github.com/mocheny-project](https://github.com/mocheny-project)  
> Artifacts: [ghcr.io/mocheny-project](https://ghcr.io/mocheny-project)

## Core Tools

- **[Mocheny CLI](https://github.com/mocheny-project/mocheny-cli)**  
  The command-line engine for Pipeline execution and State validation.

- **[Mocheny Secret](https://github.com/mocheny-project/mocheny-secret)**  
  Secure secret storage with audit trail and rotation policy.

- **[Mocheny Environment](https://github.com/mocheny-project/mocheny-environment)**  
  JPS manifests and Docker images for reproducible deployments.
